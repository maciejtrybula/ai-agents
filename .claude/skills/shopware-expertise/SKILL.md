# Agent Skill: Shopware 6 API-First Development

## Overview
Technical proficiency in Shopware 6, focusing on its API-first architecture, the App System, and Vue-based administration.

## Technical Stack
- **Language:** TypeScript / JavaScript (Frontend), PHP (Core Symfony)
- **Frameworks:** Vue.js 3, Symfony
- **Architecture:** Shopware App System (Webhook-based), Store API
- **Tooling:** Shopware CLI, PWA (Vue Storefront)

## Core Capabilities
* **App System Integration:** Developing "Apps" that interact with Shopware via webhooks and the Admin API without touching the PHP core.
* **Storefront API:** Building custom TypeScript clients to consume the Store API for headless or PWA frontends.
* **Rule & Flow Builder:** Configuring complex business automation using Shopware’s native Rule and Flow engines.
* **Admin Extensions:** Customizing the Shopware Administration panel using Vue.js 3 components and the Admin Extension SDK.
* **B2B Components:** Implementing complex B2B workflows (Employee management, Quote systems) within the Shopware ecosystem.
