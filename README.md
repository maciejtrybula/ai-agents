# ai-agents

Centralized hub for AI agent definitions and reusable skills across Claude, OpenCode, and Codex.

## Agents

### Engineering Roles

- Backend Architect - DDD, microservices design, and backend strategy
- Backend Engineer - Service/API implementation and domain logic in Node.js/TypeScript
- Frontend Architect - Web architecture, data flow, and project structure
- Frontend Engineer - UI implementation, components, and frontend testing
- UX/UI Architect - Design tokens, accessibility, and mobile-first interfaces
- DevOps Engineer - IaC, CI/CD, and cloud orchestration
- E2E Test Engineer - Playwright end-to-end testing and QA
- SecOps Auditor - Security architecture, threat analysis, and OWASP compliance

### Leadership Roles (Claude Only)

- Principal Engineer - Strategic technical leadership and cross-stack reviews
- Staff Engineer - Cross-team coordination and complex technical problem-solving
- Team Manager - People management, delivery, and organizational health

## Skills

- Code Review - Standards-based pull request and code change analysis
- Tech Arch Research - Deep technical architecture and implementation analysis

## Locations

- `.claude/agents/` - Claude agent definitions
- `.claude/skills/` - Claude skill definitions
- `.config/opencode/agents/` - OpenCode agent definitions
- `.config/opencode/skills/` - OpenCode skill definitions
- `.codex/skills/` - Codex skill definitions
