# Agent Skill: Adobe Commerce App Builder & API Mesh

## Overview
Specialization in out-of-process extensibility for Adobe Commerce (Magento) using the Adobe Developer App Builder and API Mesh.

## Technical Stack
- **Environment:** Node.js / TypeScript
- **Gateway:** Adobe API Mesh (GraphQL Orchestration)
- **Platform:** Adobe I/O Runtime (Serverless)
- **Events:** Adobe I/O Events (Real-time triggers)

## Core Capabilities
* **API Orchestration:** Configuring API Mesh to merge legacy Magento REST/GraphQL with modern microservices into a unified endpoint.
* **Out-of-Process Logic:** Building TypeScript actions in App Builder to handle business logic without modifying the PHP core.
* **Event-Driven Integration:** Subscribing to platform events (e.g., `observer.sales_order_place_after`) to trigger external syncs.
* **Admin Customization:** Developing custom UI modules using Adobe's React-based spectrum components.
* **Performance Tuning:** Optimizing GraphQL resolvers within the Mesh layer for edge-side performance.
