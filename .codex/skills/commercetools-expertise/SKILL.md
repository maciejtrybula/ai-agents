# Agent Skill: commercetools Cloud-Native Expertise

## Overview
Expertise in MACH-based (Microservices, API-first, Cloud-native, Headless) commerce orchestration using commercetools Composable Commerce.

## Technical Stack
- **Language:** TypeScript (Primary)
- **APIs:** GraphQL (Main query interface), REST (Platform API)
- **SDKs:** `@commercetools/sdk-client-v2`, `@commercetools/platform-sdk`
- **Extensibility:** API Extensions, Subscriptions (Event-driven), Custom Applications

## Core Capabilities
* **Data Modeling:** Designing and implementing Custom Types and Fields to extend the standard resource schemas.
* **API Orchestration:** Building middleware to aggregate commercetools data with external PIM or CMS providers.
* **Serverless Extensions:** Developing TypeScript-based Google Cloud Functions or AWS Lambda to intercept platform events via API Extensions.
* **Cart & Checkout Logic:** Implementing complex discount engines and tax calculations through the platform's rule-based engine.
* **State Management:** Managing complex order states and transition triggers for custom fulfillment workflows.
