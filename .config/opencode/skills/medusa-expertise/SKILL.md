# Agent Skill: Medusa.js V2 Modular Commerce

## Overview
Expertise in the open-source Medusa.js V2 framework, focusing on modular architecture and custom workflow orchestration.

## Technical Stack
- **Core:** Node.js / TypeScript
- **Architecture:** Medusa Modules, Workflows (Long-running logic)
- **Database:** PostgreSQL (with Remote Joiners)
- **Tooling:** Medusa CLI, Vite-based Admin

## Core Capabilities
* **Workflow Design:** Creating idempotent business logic using Medusa Workflows to handle multi-step commerce processes.
* **Module Development:** Building custom TypeScript modules that encapsulate specific domain logic (e.g., custom loyalty systems).
* **Data Linkage:** Using "Remote Joiners" to query data across different modules and third-party services in a single request.
* **Admin Customization:** Extending the Medusa Admin dashboard with custom React widgets and UI routes.
* **Plugin Ecosystem:** Integrating and configuring TypeScript-based plugins for payment (Stripe), search (MeiliSearch), and fulfillment.
