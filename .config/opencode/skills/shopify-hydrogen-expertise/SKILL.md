# Agent Skill: Shopify Headless & Hydrogen Expertise

## Overview
Deep expertise in building high-performance, headless commerce experiences using Shopify Hydrogen and the Storefront API.

## Technical Stack
- **Framework:** Remix / React Router v7 (latest)
- **Language:** TypeScript (Strict mode)
- **Hosting:** Shopify Oxygen (Edge-worker runtime)
- **API:** Storefront GraphQL API, Admin GraphQL API

## Core Capabilities
* **Headless Storefronts:** Developing custom buyer journeys using Hydrogen’s commerce-optimized React components.
* **Edge Performance:** Implementing advanced caching strategies (stale-while-revalidate) specifically for Oxygen’s runtime.
* **Cart Orchestration:** Managing complex client-side cart states and syncing with Shopify’s native checkout via the Storefront API.
* **Metafield Management:** Leveraging Shopify Metafields and Metaobjects for structured content delivery.
* **App Bridge Integration:** Building embedded Shopify apps using TypeScript and Polaris for merchant-facing tools.
